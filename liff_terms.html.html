第1条（適用）

本規約は、【会社名】（以下「当社」）が提供するSNS自動投稿サービス「AiPost」（以下「本サービス」）の利用に関する条件を定めるものです。

本サービスをご利用いただいた時点で、本規約に同意したものとみなします。

第2条（サービスの内容）

本サービスは、LINEを通じて以下の機能を提供します。

  1. AI文章生成機能（投稿文・ハッシュタグの自動生成）
  2. SNS予約投稿機能（Instagram・Facebook・Threads・X）
  3. 投稿分析機能（LITEプラン以上）
  4. 素材画像の提供（LITEプラン以上）
  5. その他当社が定める機能
第3条（プランと料金）

本サービスは以下のプランを提供します。料金は全て税別です。

  1. FREEプラン：無料 / SNS1つ / 月4件
  2. PERSONALプラン：月額500円 / SNS1つ / 月8件
  3. LITEプラン：月額780円 / SNS2つ / 月15件
  4. STANDARDプラン：月額1,280円 / SNS3つ / 月30件
  5. PROプラン：月額1,980円 / SNS4つ / 月60件

各プランの詳細な機能は、プラン・料金画面をご確認ください。

同じ内容を複数のSNSに同時投稿した場合は1件としてカウントします。同時投稿扱いの条件は「同一テキスト・同一画像・同時刻」の3条件が全て一致する場合に限ります。
第4条（決済・解約）
  1. 有料プランの料金はStripeを通じてクレジットカードにて月払いで決済します。
  2. プランのアップグレードは即時反映されます。
  3. ダウングレードは当月末に反映され、それまでは現在のプランをご利用いただけます。
  4. 解約は当月末に反映され、それまでは現在のプランをご利用いただけます。
  5. 日割り返金は行いません。
  6. 解約手続きは設定画面の「登録」タブから行えます。
第5条（投稿機能の利用）
  1. 利用者は、本サービスを通じて投稿する内容について全責任を負います。
  2. 各SNSの利用規約に違反する内容の投稿は禁止します。
  3. 当社は、投稿内容の正確性・適法性について保証しません。
  4. SNS側の仕様変更・API変更・障害等により投稿が失敗した場合、当社は責任を負いません。
第6条（XへのURL付き投稿の禁止）

当サービスでは、XへのURL付き投稿に対応していません。

  1. 投稿文にURLが含まれる場合、Xは自動的に投稿先から除外されます。
  2. URLの直接記述以外にも、日本語・記号・短縮URL・QRコード誘導テキスト等によるURL表現（以下「迂回行為」）も同様に禁止します。
  3. 迂回行為が確認された場合は、本規約違反として利用停止・登録解除の対象となる場合があります。
⚠️ 迂回行為とはURLを直接記述せず、他の方法でリンク先を示す行為を指します。意図的な迂回行為が確認された場合はアカウントを停止する場合があります。不明な点は事前に運営までお問い合わせください。
第7条（AI文章生成機能）
  1. AI文章生成は全プランで無制限にご利用いただけます。
  2. 生成された文章の著作権は利用者に帰属します。
  3. 当社は生成文章の品質・正確性・適法性を保証しません。
  4. 生成文章を投稿する前に必ず内容をご確認ください。
  5. 生成文章を利用したことによる損害について当社は責任を負いません。
第8条（SNSアカウントの連携と安全管理）

本サービスでは、自動投稿を行うために各SNSが発行する「アクセストークン」をお預かりします。

アクセストークンとは？
各SNSが発行する「投稿専用の許可証」です。通常のパスワードとは全く異なるもので、以下の性質があります。
・投稿する権限のみを持ちます
・ログインや個人情報の閲覧には使えません
・利用者がいつでも無効にできます
  1. 暗号化による安全管理
    お預かりするアクセストークンはAES-256という業界標準の暗号化方式で保存されます。保存されたデータは意味不明な文字列に変換されるため、運営者を含む第三者が直接内容を読むことは技術的にできません。
  2. 当社が預かるもの・預からないもの
    【預かるもの】投稿専用のアクセストークン（暗号化済み）
    【預からないもの】SNSのパスワード・個人情報・金融情報
  3. パスワードについて
    当社はSNSのパスワードを一切保持していません。パスワードを忘れた場合は各SNSのサポートにお問い合わせください。当社では対応できません。
  4. アクセストークンの取り消し
    各SNSの設定画面からいつでも連携を解除・アクセストークンを無効にすることができます。
  5. 免責
    当社は適切なセキュリティ管理を行いますが、各SNS側のシステム障害・仕様変更、または利用者自身のアカウント管理に起因する問題については対応できない場合があります。
第9条（素材画像）
  1. 素材画像はLITEプラン以上でご利用いただけます。
  2. 素材画像は本サービス内での投稿用途のみに限定されます。
  3. 素材画像のダウンロード・外部への転用は禁止します。
  4. 素材画像の著作権は各権利者に帰属します。
第10条（匿名統計データの利用）

本サービスでは、PROプランの競合比較機能のために、利用者の投稿分析データを匿名化した上で統計処理に使用します。個人を特定できる情報は一切含まれません。詳細はプライバシーポリシーをご確認ください。

利用者は以下の行為を行ってはなりません。

  1. 法令または公序良俗に違反する行為

  3. XへのURL付き投稿の迂回行為
  4. 当社または第三者の知的財産権を侵害する行為
  5. 本サービスの運営を妨害する行為
  6. 不正アクセスその他のセキュリティに反する行為
  7. 複数アカウントの作成・譲渡
  8. その他当社が不適切と判断する行為
第12条（利用停止・登録解除）
  1. 当社は、利用者が禁止事項に違反した場合、事前通知なくサービスの利用停止・登録解除を行うことができます。
  2. 登録解除された場合、投稿予約・分析データ等は削除されます。
  3. 登録解除に伴う返金は行いません。
第13条（免責事項）

  2. SNS側のAPI変更・仕様変更・障害により投稿が失敗した場合、当社は責任を負いません。
  3. AI生成文章の内容・品質について保証しません。
  4. 本サービスの利用により生じた損害について、当社の故意または重大な過失がある場合を除き、責任を負いません。
第14条（規約の変更）

当社は必要に応じて本規約を変更することができます。変更後の規約は本サービス内に掲載した時点で効力を生じます。変更後も本サービスをご利用いただいた場合、変更後の規約に同意したものとみなします。
